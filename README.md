# Library

プログラミングコンテスト(Codeforces, ICPC等)で使用する為のコード断片集です

C++11実装で、たまにD言語実装もあります

## 内容(Cpp)

### データ構造

- Union Find
- Fenwick Tree, Fenwick Tree 2D
- Starry Sky Tree
- Wavelet Tree?
- Dynamic Segtree(AA Tree)
- Multiset(AA Tree)
- HL Decomposition
- Link Cut Tree
- Sparse Table
- Radix Heap
- さいころ
- 行列

### グラフ

- Dijkstra
- LCA
- SCC, 2SAT
- Maxflow(Dinic)
- Mincostflow
- Lowlink, Bridge
- 後退解析
- Dominator Tree

### 数学

- FFT

### 文字列

- KMP, Manachar, Z
- RollingHash
- SuffixArray(O(Nlog^2N), O(N)), LCP

### 幾何

- 各種衝突判定
- 線分アレンジメント
- 双対グラフ
- 凸法
- 最小包含円
